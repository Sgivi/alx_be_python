# Control Flow & Loops – Python Exercises

This directory contains exercises that demonstrate the use of conditional
statements, match case statements, loops, and nested loops in Python.

## Files Included

### 1. weather_advice.py

A program that gives clothing recommendations based on weather input using
if/elif/else statements.

### 2. match_case_calculator.py

A simple calculator that uses Python's match case pattern to select operations.

### 3. multiplication_table.py

Generates a multiplication table (1–10) for a number using a for loop.

### 4. pattern_drawing.py

Uses a while loop + nested for loop to draw a square pattern of asterisks.

### 5. daily_reminder.py

A user task reminder system that uses match case and conditionals to produce
custom reminders based on priority and time sensitivity.

## Purpose

These tasks reinforce:
- Control flow
- Conditional logic
- Pattern matching
- For loops
- While loops
- Nested loops

Core skills for backend Python development.
